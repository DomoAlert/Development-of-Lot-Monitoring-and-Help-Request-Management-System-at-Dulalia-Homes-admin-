# Dulalia Homes Admin System - User Manual

## üìã Introduction

### System Overview
The **Dulalia Homes Admin System** is a comprehensive web-based administrative dashboard designed specifically for managing residential communities at Dulalia Homes. This powerful tool provides property administrators, managers, and staff with complete control over community operations, resident services, and facility management.

### System Type
Web-based administrative application built with React.js

### Target Users
- Property administrators and managers
- Community staff and coordinators
- Security personnel supervisors
- Maintenance and service coordinators
- Administrative assistants

### Purpose of the System
The Dulalia Homes Admin System serves as the central hub for managing all aspects of residential community operations, including:
- Real-time monitoring of community metrics and activities
- Managing resident accounts and property assignments
- Processing service requests and maintenance issues
- Coordinating facility bookings and usage
- Managing security personnel and visitor access
- Publishing community announcements
- Tracking inventory and community assets
- Collecting and responding to resident feedback

## üìñ How to Use the System

### Core Functions Guide

#### 1. Managing Service Requests
**Step-by-step process:**
1. Navigate to "Service Requests" from the sidebar
2. Review pending requests in the table
3. Click on a request to view details
4. Add comments or notes as needed
5. Update request status (Pending ‚Üí In Progress ‚Üí Completed)
6. Notify resident of status changes

#### 2. Processing Facility Bookings
**Step-by-step process:**
1. Go to "Facility Requests" section
2. Review pending facility booking requests
3. Check facility availability and resident eligibility
4. Approve or reject requests with appropriate reasoning
5. Send confirmation notifications to residents

#### 3. Creating Announcements
**Step-by-step process:**
1. Access "Announcements" from the navigation menu
2. Click "Create Announcement" button
3. Fill in announcement details:
   - Title and content
   - Target audience (Everyone/Homeowners/Guards)
   - Announcement type (Notice/Update/Urgent)
   - Expiration date
4. Review and post the announcement

#### 4. Managing User Accounts
**Step-by-step process:**
1. Navigate to "User Accounts" section
2. Search or filter users as needed
3. Click on a user to view/edit details
4. Update property assignments if necessary
5. Modify account status or permissions
6. Save changes and notify user if required

#### 5. Monitoring Lots and Properties
**Step-by-step process:**
1. Access "Lot Monitoring" from the sidebar
2. View current lot statuses and occupancy
3. Update lot information as needed
4. Track property assignments
5. Generate reports on lot utilization

## üîß Troubleshooting

### Common Issues and Solutions

#### Login Problems
**Issue**: Cannot log in to the system
**Solutions**:
- Verify email address and password are correct
- Check if Caps Lock is enabled
- Clear browser cache and cookies
- Try a different web browser
- Contact administrator if account is locked

**Issue**: "Invalid credentials" error
**Solutions**:
- Double-check email and password spelling
- Reset password if forgotten
- Ensure account is active and not suspended
- Contact system administrator for account verification

#### Page Loading Issues
**Issue**: Pages take too long to load or don't load at all
**Solutions**:
- Check internet connection stability
- Refresh the page (Ctrl+F5 for hard refresh)
- Clear browser cache and cookies
- Disable browser extensions temporarily
- Try accessing from a different network

#### Data Not Updating
**Issue**: Changes not appearing immediately
**Solutions**:
- Wait a few moments for real-time updates
- Refresh the page manually
- Check browser console for error messages
- Clear browser cache if issues persist

#### Form Submission Errors
**Issue**: Cannot submit forms or save changes
**Solutions**:
- Ensure all required fields are filled
- Check for validation error messages
- Verify data format (dates, phone numbers, etc.)
- Try submitting again after a moment

#### File Upload Issues
**Issue**: Cannot upload images or documents
**Solutions**:
- Check file size limits (usually 5-10MB)
- Ensure correct file format (JPG, PNG, PDF)
- Verify stable internet connection
- Try a different file or smaller size

## ‚ùì FAQs

### Getting Started
**Q: How do I get access to the admin system?**
A: Contact your Dulalia Homes administrator or manager. They will create your account and provide login credentials.

**Q: What should I do if I forget my password?**
A: Use the "Forgot Password" link on the login page. Enter your email address and follow the reset instructions sent to your email.

**Q: Can I access the system from my mobile phone?**
A: Yes, the system is responsive and works on mobile devices, but some features work better on desktop or tablet devices.

### Using the System
**Q: How do I know if there are urgent requests that need attention?**
A: Check the dashboard for pending request counts and status indicators. Urgent items are usually highlighted in red or marked with exclamation icons.

**Q: Can I edit or delete announcements after posting them?**
A: Yes, you can edit announcements from the Announcements page. However, deleted announcements cannot be recovered, so be careful.

**Q: How do I assign properties to residents?**
A: Go to User Accounts, find the resident, click "Edit", and use the property assignment section to link them to specific lots.

### Technical Questions
**Q: What browsers are supported?**
A: The system works best with Chrome, Firefox, Edge, and Safari (latest versions). Internet Explorer is not supported.

**Q: Is my data secure?**
A: Yes, the system uses Firebase authentication and encrypted data storage. All communications are secured with SSL/TLS encryption.

**Q: Can multiple administrators use the system at the same time?**
A: Yes, the system supports multiple concurrent users with appropriate access controls.

**Q: How often is data backed up?**
A: Data is automatically backed up through Firebase's infrastructure. Contact your administrator for specific backup schedules.

### Feature-Specific Questions
**Q: How do I generate visitor QR codes?**
A: Go to Visitor Logs, click "Generate QR Code", fill in visitor details, and the system will create a scannable QR code.

**Q: Can I export reports or data?**
A: Yes, most sections have export options. Look for "Export" or "Download" buttons in tables and chart sections.

**Q: How do I handle emergency situations?**
A: Use the emergency contact features, create urgent announcements, and coordinate with security personnel through the system.
